Usage Instructions

How to Use Gradient Buttons

Simply add the gradient class to your existing button elements:

<!-- Primary gradient button --> <button class="btn btn-primary btn-gradient-primary"> <i class="fas fa-users"></i> Join the Movement </button> <!-- Saffron gradient button --> <button class="btn btn-secondary btn-gradient-saffron"> <i class="fas fa-heart"></i> Donate Now </button> <!-- Success gradient button --> <button class="btn btn-gradient-success"> <i class="fas fa-check"></i> Register </button>

Available Gradient Classes

  • .btn-gradient-primary - Blue to Green political gradient
  • .btn-gradient-saffron - Saffron to Gold traditional gradient
  • .btn-gradient-success - Green success gradient

CSS Variables Used

--gradient-primary: linear-gradient(135deg, #0266B4 60%, #008E46 100%); --gradient-saffron: linear-gradient(135deg, #FF9933 0%, #FFD700 100%); --gradient-success: linear-gradient(135deg, #10b981 0%, #047857 100%);
